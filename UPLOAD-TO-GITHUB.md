# Upload instructions

This folder is the public-safe source package for `Gudarzian/FederationAutomation`.

1. Open https://github.com/Gudarzian/FederationAutomation in a browser and sign in.
2. Select **Add file** then **Upload files**.
3. Drag the contents of this folder into the upload area. Upload the folders `Scripts` and `Docs` as folders, together with `README.md`, `.gitignore`, and `LICENSE`.
4. Confirm that `Scripts/Config.json`, `Scripts/Config.xlsx`, EXE files, logs, output folders, backups, and `NavisworksOptions.xml` are not in the upload list.
5. Enter the commit message: `Initial public release`.
6. Select **Commit directly to the main branch** and commit.
7. Open the repository home page and confirm that the README and LICENSE are visible.

Do not upload this `UPLOAD-TO-GITHUB.md` file unless you want these instructions to be public.
